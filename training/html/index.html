A11y

There are a number of good a11y tools out there. One of my favorites is tota11y.

The A11y Project is a great source of tools and tutorials.

  • VoiceOver is a native screen reader for Mac and iOS.
  • TalkBack is a native screen reader for Android.
  • Narrator is a native screen reader for Windows 10.
